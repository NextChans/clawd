# Screenshots

Drop the images the root `README.md` links to here. Suggested set (PNG, retina
if you can):

| File               | What to capture                                             |
|--------------------|-------------------------------------------------------------|
| `roam.png`         | The cat wandering the desktop in 🐾 Roam mode.              |
| `grab.png`         | 🖐️ Grab mode — glowing ring + hover tooltip with stats.     |
| `details.png`      | The details window: cards, donut, sparkline, heatmap.       |
| `furniture.png`    | On-demand furniture (cushion / tower / bowl) + a plaything. |
| `tray.png`         | The menu-bar tray menu.                                      |

To grab one: run the app (`npm run tauri dev`), pose the cat, and use
**⌘⇧4** (drag) or **⌘⇧4 then Space** (window) to capture. Keep them reasonably
sized (~1200px wide is plenty) so the repo stays light.
